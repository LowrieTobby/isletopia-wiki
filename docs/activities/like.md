# NameMC点赞

::: tip 提示
在NameMC为服务器点赞可以吸引更多海外玩家，让社区更加多元化。
:::

* 在NameMC为服务器点赞需要正版账号
* NameMC账号需要使用非qq邮箱注册
* 点赞后可使用`/likereward`指令领取龙首奖励一次
* 每个正版账号最多领取一次，更换ID后不能再次领取。
* 如果你不想给服务器点赞，可以领取奖励后取消点赞。
